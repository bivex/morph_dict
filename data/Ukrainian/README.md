# Ukrainian Morphological Dictionary Data

This directory contains source data for building the Ukrainian morphological dictionary.

## Required Files
- `gramtab.json` - Grammar table with Ukrainian grammatical categories
- `morphs.json` - Morphological paradigm data (large file, generated by morph_wizard)
- `project.mwz` - Project configuration for morph_wizard
- `StatData.txt` - Statistical data for homonym disambiguation weights
- `WordData.txt` - Word frequency data for ranking

## Getting the Data
The Ukrainian dictionary data is not included in the repository. To obtain or create it:
1. Collect a large corpus of Ukrainian text (millions of words)
2. Use `morph_wizard` tool to process the corpus and generate `morphs.json` and other files
3. Create appropriate `gramtab.json` for Ukrainian grammatical categories
4. Generate statistical weight files using `StatDatBin` and `word_freq_bin`

See the `morph_wizard` documentation for details on dictionary creation.

## Building
Once data files are in place, the build system will automatically generate binary dictionary files in `Dicts/Morph/Ukrainian/`:
- `morph.bases`
- `morph.forms_autom`
- `morph.annot`
- `gramtab.json` (copied)
- `morph.options`
- Weight files: `chomoweight.bin`, `cwordweight.bin`, `fhomoweight.bin`, `fwordweight.bin`, `lhomoweight.bin`, `lwordweight.bin`
- `npredict.bin`
